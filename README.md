# Novel Reader

A mobile-friendly static web reader for translated novel chapters, published via GitHub Pages.

## Live Website

**[https://augustolamim.github.io/novel-reader/](https://augustolamim.github.io/novel-reader/)**

## Project Structure

```text
novel/
  index.html
  styles.css
  app.js
  chapters.json
  chapters/
```

## How to Update

Update the files at the repository root and push to `main` to redeploy the site.
